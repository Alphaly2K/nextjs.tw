什麼是編譯?

通常開發者們使用 JSX、TypeScript 與 JavaScript 等這些對他們較為親近的程式語言撰寫程式碼。

雖然這些語言提高開發者們的信心與效率，不過它們依然需要先編譯成 JavaScript 後，瀏覽器才能解析並理解這些程式碼。

![](https://nextjs.org/static/images/learn/foundations/compiling.png)

編譯是指以一種程式語言撰寫程式碼後，使用另一種語言或以此語言的其他版本進行輸出的過程。

在 Next.js 中，編譯會發生您對程式碼進行編輯時，即開發階段；以及準備打包你的應用程式至正式環境前的部分程序。
